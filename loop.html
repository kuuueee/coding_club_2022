Intro to For Loops


Wouldn't it be nice to be able to be able to interact with each of the elements one by one?

With for loops we are able to achieve things such as a search, or a print all

In Javascript a for loop will look like this:

    for (var k = 0; k < 9; k++){
        //output the value of k;
    }

In english this for loop says:

  • var k = 0;  → variable named k is equal to zero
  • k < 9;   → while k is less than 9
  • k++    → add 1 to k at the end of the loop.

What do you think this for loop will do?
