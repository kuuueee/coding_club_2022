Intro to Conditional statements, If, Else

Imagine a way for computers to make decisions. How would that work?

Take this stop light for example:






How does this computer decide what lights to changes?


With a conditional statement called: If

        
        If  No    -> then Change to Red
(else)  if  Maybe -> then Change to Yellow
(else)  if  Yes   -> then Change to Green
(else)      None of the above -> then Change all to grey
        
        
        Another way to express this is:
        
        if (action == no){
            //change to red
        } else if (action == maybe){
            //change to yellow
        } else if (action == no){
            //change to no
        } else {
            //change all to grey
        }
